## Text Content
This section deals with program capabilities development and acquisition weapon system design:

"Requirement to address MOSA in program capabilities development and acquisition weapon system design."

(a) In General.—Each Major Defense Acquisition Program (MDAP) shall—
(1) include an assessment of the program’s ability to implement a modular open system; and
(2) be designed and developed to the maximum extent practicable using a modular open system;

(b) Program Capabilities Development.—In the program capabilities development of each major defense acquisition program, the Secretary of Defense shall ensure—
(1) the program’s capability to implement a modular open system is assessed; and
(2) the program is designed and developed using a modular open system to the maximum extent practicable.

(c) Weapon System Design.—In the weapon system design of each major defense acquisition program, the Secretary of Defense shall ensure—
(1) the weapon system is designed and developed using a modular open system to the maximum extent practicable; and
(2) the weapon system is designed and developed to incorporate modular system interfaces that—
(A) are defined in accordance with widely known or publicly available standards;
(B) are provided with machine-readable documentation;
(C) are sufficient to enable competition;
(D) are sufficient to enable interoperability; and
(E) are sufficient to support technology refresh.

## Key Provisions
- **Assessment Requirement**: Each MDAP must assess its ability to implement MOSA
- **Design Requirement**: Must be designed and developed using MOSA "to the maximum extent practicable"  
- **Weapon System Specific**: Must incorporate MOSA interfaces with specific characteristics:
  - Based on widely known/public standards
  - Machine-readable documentation
  - Enable competition
  - Enable interoperability  
  - Support technology refresh
