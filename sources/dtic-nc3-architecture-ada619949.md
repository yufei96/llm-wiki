# GAO-15-584R — NC3 现代化更新

> 来源摘要页面。原始文本: [[raw/articles/dtic-nc3-architecture-ada619949]]

## 核心内容

### 报告背景

本报告应参议院军事委员会（SASC）FY2015 NDAA 报告要求发布，属于 GAO 自2013年以来对NC3进行的系列审查的一部分。审查目标：
1. 更新此前对NC3采办项目的评估
2. 审查美国战略司令部 (STRATCOM) 更新任务规划与分析系统的工作

### 审查的 7 个 NC3 项目

GAO 重点审查了7个关键NC3项目/工作：

1. **任务规划与分析系统** (Mission Planning and Analysis System)
2. **先进超视距终端系列** (FAB-T)
3. **总统与国家语音会议** (Presidential and National Voice Conferencing)
4. **凤凰空对地通信网络** (Phoenix Air-to-Ground Communications Network)
5. **通用甚低频接收机** (Common Very Low Frequency Receiver)
6. **全球空勤战略网络终端** (Global Aircrew Strategic Network Terminal)
7. **民兵最低限度应急通信网络计划升级** (Minuteman MEECN Program Upgrade)

选择标准：按估算成本和/或支撑高级指挥官通信或其他关键能力的重要性排名最大的项目。

### 审查内容

对每个项目评估了三个方面：
- **采办策略**：开发和部署方式，如何应对已知能力缺口
- **进展**：成本、进度和绩效目标的满足情况
- **风险**：满足目标的风险及缓解措施

### 主要发现

- 7个NC3项目/工作在开发和部署方面**取得了进展**
- 但**已知能力缺口仍然存在**：
  - 战略轰炸机关键卫星通信终端的开发被推迟数年
  - 部分项目的后续增量尚未获得资金
- 更多详细发现仍属**机密**

### NC3 架构特征

- NC3 是一个大型复杂系统，包含众多**陆基、空基和天基组件**
- 确保总统与核力量之间的连通性
- 既支持日常核/常规作战，也提供可生存、安全、持久的通信
- 大多数NC3系统同时支持核和常规任务

### 方法论

- 审查期：2014年10月至2015年6月
- 审查方式：文件审查、项目官员访谈
- 访谈对象包括：空军战略威慑与核整合办公室、OSD(AT&L)、STRATCOM、DOT&E、CAPE、DISA、AFGSC、JHU/APL等
