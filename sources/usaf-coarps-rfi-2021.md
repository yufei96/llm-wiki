# USAF COARPS RFI — AWACS Common Open Architecture Radar Programs (2021)

## 核心内容

- **COARPS** = Common Open Architecture Radar Programs，美国空军开发的**开放雷达架构规范**
- 发布机构：E-3 Integration Branch, **AFLCMC/HBSA**, Hanscom AFB, MA
- RFI 目的：识别正在开发/集成 COARPS 模式的行业合作伙伴
- COARPS 设计目标：
  - **独立采购雷达子系统**（不受单一供应商锁定）
  - **处理器硬件刷新无需大规模软件重写**
  - **第三方模式技术插入**（third-party mode insertion）
  - **缩短雷达模式采购时间线**
- 2019年，空军启动多项 **ABMS GMTI**（地面移动目标指示）项目，聚焦 COARPS 兼容 GMTI 模式开发
- 重点方向：路径发现第三方模式开发、为未来 ABMS 系统演进 COARPS 标准、建立可移植 GMTI 模式、建立 AF 雷达模式存储库
- RFI 询问两个核心问题：
  1. 是否有 COARPS 2.0 模式开发/集成经验
  2. 是否计划未来 COARPS 2.0 模式开发
