# CRS IF11681 — LGM-35A Sentinel 洲际弹道导弹入门

> 来源摘要页面。原始文本: [[raw/articles/crs-sentinel-icbm-if11681]]

## 核心内容

### 项目概述

LGM-35A Sentinel 是预计替换民兵 III (MMIII) 的新一代洲际弹道导弹系统。MMIII 自1970年以来一直是美国核三位一体的陆基支柱。国防部使用"战争部"二级指定（依据2025年9月5日第14347号行政命令）。

### 经费与立法

| 指标 | 数据 |
|------|------|
| FY2026 RDT&E 申请 | **$41亿** |
| FY2025 调和法拨款（P.L. 119-21） | **$25亿** 风险降低活动 |
| FY2026 NDAA 授权 RDT&E | **$38亿** |
| W87-1 核弹头（NNSA） | **$6.49亿** |
| CAPE 重新估算总成本 | **$1410亿**（"合理修改后"） |
| 每枚导弹单位成本增幅 | ≥37%（Nunn-McCurdy 违规） |

### ICBM 基本参数

- 射程：全球覆盖，发射后约 **30分钟** 抵达目标
- 推进段：约3分钟，三级固体燃料火箭
- 中段：弹道飞行，释放弹头
- 一旦授权发射，**不可召回**或在飞行中摧毁
- 美国自1959年起持续保持ICBM战备状态

### 从 MMIII 过渡

- MMIII 1970年入役，单弹头构型（可携带3枚）
- 2014年替代方案分析后决定替换
- 2029年原定开始替换MMIII（已推迟）
- 服役目标：**2075年**

### 采办数量与部署

- **634枚** Sentinel 导弹（含25枚研发测试用）
- **400枚** 部署
- 原计划现代化 **450个发射井**及600+设施
- 部署基地：F.E. Warren、Malmstrom、Minot AFB（北达科他、蒙大拿、怀俄明、科罗拉多、内布拉斯加）

### Nunn-McCurdy 违规与重组

- **2024年1月**：告知国会成本超支≥37%（从$1.18亿/枚增至$1.62亿/枚，2020美元）
- **2024年7月**：完成Nunn-McCurdy审查，Milestone B 批准被撤销
- 空军正在重新审查需求，使地面段"更简单"和"更可负担"
- **2025年12月**：参议院确认 Dale White 将军为直接报告项目组合经理 (DRPM)
- **2026年2月**：
  - 预计2026年底前完成项目重组并达成 Milestone B
  - 初始能力目标：**2030年代初**
  - 改为建造新发射井（非复用旧井），需补充环境影响报告
  - 陆军工程兵团（非Northrop Grumman）管理部分地面基础设施
  - 2027年进行垫发射，GAO报告首飞测试推迟至 **2028年3月**

### 弹头

- 初始部署使用 MMIII 的 **W87-0** 弹头
- **W87-1** 新弹头：NNSA 研发，预计 **FY2031-FY2032** 部署
- 2024年10月完成首个作战储备钚弹芯鉴定
- Lockheed Martin 研发 Mk21A 再入飞行器

### 开放架构设计

Sentinel 采用**开放系统架构** (Open System Architecture)，旨在：
- 模块化设计，便于生命周期改进
- 支持组件级升级而无需更换整个系统
- 降低全生命周期成本

> 另见 [[open-architecture-hierarchy]] 了解开放架构在Sentinel项目中的具体应用。

### 国会关注议题

1. **ICBM 力量必要性**：支持/反对论点；2022 NPR 确认三位一体互补
2. **MMIII 延寿替代**：空军2024年表示看不到"可行的"长期延寿方案；GAO 2025年9月评估MMIII可服役至2050年但面临零部件过时风险
3. **成本与进度**：Nunn-McCurdy 违规；FY2025 NDAA §1629 设定条件要求
4. **过渡风险**：
   - 法律要求至少 **400枚** 战备ICBM
   - FY2026 NDAA 要求400枚ICBM平均分布在≥450个发射井中
   - MMIII 与 Sentinel 将重叠运营**至少15年**
   - GAO 2025年9月报告建议制定过渡风险管理计划
5. **多弹头与机动性**：2023年战略态势委员会报告提议考虑多弹头和公路机动构型
