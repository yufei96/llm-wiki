# AFGSC — Sentinel 部署与民兵III退役处置

> 来源摘要页面。原始文本: [[raw/articles/afgsc-sentinel-gbsd]]

## 核心内容

### 项目定义

空军全球打击司令部 (AFGSC) 的 Sentinel 部署项目（前称"陆基战略威慑力量"GBSD）旨在替换美国本土部署的所有陆基 MMIII ICBM。

### 替换范围

所有 MMIII 导弹组件均将被替换：
- 三级固体燃料火箭发动机
- 两个级间段
- 动力系统火箭发动机
- 导弹制导组件
- **核弹头数量、尺寸、构型和设计不变**（由能源部提供）

### 联邦采购管理

- 空军使用**陆军工程兵团 (USACE)** 作为采购代理
- USACE 不动产专家负责联系土地所有者
- 流程：评估→确定公平市场价值→提出补偿报价→获取必要权益

### 环境许可与合规

- 项目包含**环境影响报告书 (EIS)**
- CRS IF11681（2026年2月）指出：改为建造新发射井需要**补充环境影响报告**
- 项目涵盖环境分析、土地测量、评估、建设地点审查

### 土地征用类型

| 用途 | 临时/永久 |
|------|-----------|
| 光纤通信线安装 | 临时 |
| 建设堆放场 | 临时 |
| 新光纤线路地役权 | 永久 |
| 新发射井 | 永久 |
| Sentinel 通信塔 | 永久 |

### 最新动态（截至2026年4月）

- 341导弹大队首次使用有效载荷运输车替代品执行作战任务
- Sentinel 特遣队第10分队在F.E. Warren AFB推进ICBM现代化
- 空军领导层访问Minot AFB推进Sentinel现代化
- GT 255：ICBM试射验证多弹头再入飞行器及系统可靠性
