# 空军全球打击司令部（AFGSC）

## 摘要

美国空军全球打击司令部（Air Force Global Strike Command，AFGSC）是美国空军下属的一级司令部（Major Command），负责空军的全部战略核力量——包括陆基洲际弹道导弹（ICBM）部队和战略轰炸机部队。AFGSC成立于2009年8月7日，2019年又从空军装备司令部（AFMC）接管了ICBM相关职能，成为陆基核威慑的唯一管理司令部。在MOSA语境下，AFGSC是[[Sentinel洲际导弹政府参考架构|Sentinel GRA]]（新型ICBM）的主要使用方，也是核现代化中开放系统架构推进的关键执行机构。

## 组织背景

### 成立与历史沿革

AFGSC的建立源于2008年美国空军核武器相关事故（2007年B-52误携核弹头飞行事件和2008年ICBM引信部件误运台湾事件）后对核力量管理的全面整改。2008年国防部长Robert Gates决定将核力量职责集中化，空军于2009年8月7日在路易斯安那州巴克斯代尔空军基地成立AFGSC，最初编入第8航空队（轰炸机）和第20航空队（ICBM）。

- **2009年**：AFGSC正式成立，接管原空军作战司令部（ACC）的战略轰炸机和原空军太空司令部（AFSPC）的ICBM部队
- **2015年**：第8航空队（8 AF）转隶至AFGSC，轰炸机部队全面整合
- **2019年**：原由空军装备司令部（AFMC）管理的ICBM系统采办和维护职能（包括ICBM系统管理局、导弹维护大队等）全部转入AFGSC，实现从采办到作战的ICBM全链条统一管理
- **2020年后**：AFGSC开始主导Sentinel（LGM-35A）替代Minuteman III的全面现代化项目

### 现任领导

AFGSC司令官通常为空军上将（General），向空军参谋长（CSAF）汇报，同时在作战指挥链中向[[战略司令部|美国战略司令部（US STRATCOM）]]提供核力量。

## 核心职责

AFGSC的核心使命是维护和运行美国的空基和陆基战略核力量，涵盖三大核心职责：

### 陆基核威慑（ICBM）

AFGSC运营美国全部450个Minuteman III发射井，部署于三个ICBM联队：
- **第90导弹联队**（F.E. Warren AFB，怀俄明州）
- **第341导弹联队**（Malmstrom AFB，蒙大拿州）
- **第91导弹联队**（Minot AFB，北达科他州）

当前运营LGM-30G Minuteman III（自1970年服役），正在推进[[Sentinel洲际导弹政府参考架构|LGM-35A Sentinel]]全面替换。AFGSC也负责Minuteman III的日常维护、警报响应和作战值班。每个导弹联队下辖多个导弹中队，每个中队管理50枚导弹，24/7/365保持戒备状态。

### 战略轰炸机

AFGSC运营美国空军的全部战略轰炸机力量，包括：
- **B-52H Stratofortress**（第2轰炸机联队、第5轰炸机联队）
- **B-2 Spirit**（第509轰炸机联队，怀特曼AFB）
- **B-1B Lancer**（部分，逐步退役中）
- **B-21 Raider**（下一代隐形轰炸机，正在测试，计划2030年代初期形成初始作战能力）

轰炸机部队提供核常兼备的全球打击能力——所有轰炸机均可执行常规任务，B-52和B-2同时保持核警戒能力。第8航空队作为轰炸机组成司令部，负责轰炸机部队的作战指挥和训练。

### 战略预警与核指挥控制

AFGSC运行和维护部分核指挥控制与通信（NC3）基础设施，包括：
- 固定和移动发射控制中心（LCC）
- 应急火箭通信系统（ERCS）
- 空中发射控制系统（ALCS，由E-6B Mercury上的空射控制中队操作）
- 导弹预警传感器和攻击评估系统接口

AFGSC的NC3职能与[[NC3-GRA核指挥控制|NC3体系]]高度耦合——所有ICBM发射命令必须经过总统授权并通过NC3链路传达至AFGSC的发射控制中心。

## Sentinel LGM-35A 项目

### 项目概述

LGM-35A Sentinel（前称GBSD——Ground Based Strategic Deterrent）是AFGSC主导的下一代洲际弹道导弹项目，目标全面替换自1970年服役的LGM-30G Minuteman III。这是50多年来ICBM部队最大规模的现代化，也是美国核三位一体陆基支柱的全面更新。

### 关键数据

| 指标 | 数据 |
|------|------|
| 导弹采购总数 | 634枚（开发测试25枚，部署400枚） |
| 发射井 | 450个（新建，非复用旧井） |
| 部署基地 | F.E. Warren、Malmstrom、Minot AFB |
| 主承包商 | Northrop Grumman |
| 作战寿命 | 至2075年 |
| 设计服役期 | 50年+ |
| CAPE总成本估计 | $141B |
| FY2026研发预算 | $4.1B |
| 初始作战能力 | 2030年代初期 |

### 技术特点

- **三级固体燃料火箭发动机**，发射井发射
- **载荷**：W87-0（初始300千吨）→ W87-1（475千吨）核弹头，Mk21A再入飞行器
- **数字工程驱动**：全生命周期数字模型支撑设计、验证和维护
- **模块化开放系统架构（MOSA）**：核心设计哲学，支持至2075年的持续升级
- **地面段简化**：Nunn-McCurdy违规后重新审视需求，新建发射井方案

### 项目挑战

Sentinel项目面临重大成本和进度挑战：
- **2024年1月**：至少37%成本超支（从$1.18亿/枚到$1.62亿/枚），触发Nunn-McCurdy"严重"违规审查
- **里程碑B被撤销**：项目全面重组，重新评估需求
- **并行运行挑战**：MMIII与Sentinel至少15年并行运行，需同时维持两套系统
- **单供应商风险**：Boeing于2019年退出竞争，Northrop Grumman成为唯一主承包商
- **国会持续监督**：GAO多次审查，要求制定过渡风险管理计划

## MOSA关联：核现代化中的开放标准推进

AFGSC是MOSA（模块化开放系统架构）从战术航空领域扩展到战略核力量领域的核心试验场。

### Sentinel GRA与MOSA

[[Sentinel洲际导弹政府参考架构|Sentinel GRA]]是AFGSC将MOSA原则落地的顶层框架，定义ICBM系统的模块化开放架构：
- **模块化设计**：导弹制导系统、再入飞行器接口和发射控制系统均实施模块化接口定义
- **开放接口**：Northrop Grumman合同明确要求遵守Sentinel GRA定义的开放接口
- **长期适应性**：目标服役至2075年，需要极高的架构前瞻性
- **数字工程**：DoDI 5000.97数字工程替代传统文档化，在安全约束下实现架构透明化

### 核域MOSA的特殊约束

与常规武器系统不同，核领域中MOSA的实施面临独特限制：
1. **安全分类限制**：多数接口规范无法公开，直接制约开放标准的使用
2. **生存性优先**：加固、抗毁冗余和电磁脉冲防护比模块化替换更重要
3. **供应链封闭**：核安全要求限制了供应商数量和竞争强度
4. **长生命周期**：50年+的系统需要极高的接口前瞻性

### 推进策略

尽管存在约束，AFGSC仍在多个维度推进MOSA：
- **GRA架构框架**：通过政府掌握架构定义权，在有限范围内促进竞争
- **分层接口策略**：在非安全敏感层引用OMS/UCI等开放标准，安全层维持专有加固设计
- **渐进竞争引入**：陆军工程兵团接管地面基础设施建设、Sierra Nevada Corp.获得SAOC合同——在核域探索有限竞争模式

## 与STRATCOM的关系

AFGSC作为美国空军的一级司令部，与[[战略司令部|美国战略司令部（US STRATCOM）]]保持密切的作战指挥关系：

### 作战指挥链

- **日常管理**：AFGSC归空军参谋长（CSAF）行政管辖，负责核力量的训练、维护和后勤
- **作战指挥**：当总统或国防部长下达核作战命令时，AFGSC的核力量受US STRATCOM作战指挥（Combatant Command, COCOM）
- **双重角色**：AFGSC司令官同时担任US STRATCOM的空军组成司令部指挥官（CCMD），在战略司令部的核作战计划中扮演关键角色

### 任务协调

- **ICBM作战**：STRATCOM制定核作战计划（OPLAN 8010-12），AFGSC执行ICBM发射的具体操作
- **轰炸机任务**：轰炸机部队在常规任务中受空军作战司令部（ACC）指挥，在核任务中受STRATCOM指挥
- **NC3集成**：AFGSC的发射控制系统与STRATCOM拥有的NC3体系紧密集成——STRATCOM通过E-6B Mercury和E-4B NAOC传达总统命令，AFGSC执行

### 现代化配合

- Sentinel/GRA现代化由AFGSC主导实施，STRATCOM作为主要需求方和使用方提供作战需求
- NC3现代化中，AFGSC负责导弹相关的NC3组件更新，STRATCOM负责顶层架构协调
- 两者在FY2024 NDAA建立的主力量项目（Unified MFP）中共同参与NC3预算管理
