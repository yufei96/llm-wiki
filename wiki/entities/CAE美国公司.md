# CAE USA

## 摘要

CAE USA是加拿大CAE公司（全球模拟训练龙头）在美国的子公司，也是美国空军SCARS（模拟器通用架构需求标准，Simulators Common Architecture Requirements and Standards）项目的主承包商。SCARS旨在为美军约2400台模拟器（覆盖50+平台、300个站点、7个主要司令部）建立基于模块化开放系统方法（MOSA）的下一代模拟训练体系。

## 公司背景

### 全球模拟训练龙头

CAE Inc.（加拿大航空电子设备公司）成立于1947年，总部位于加拿大蒙特利尔，是全球最大的民航飞行模拟器制造商和军事训练系统集成商之一。公司在全球30+国家设有业务，主营业务覆盖三大板块：

- **民航模拟训练**：全球约70%的民用飞行模拟器市场由CAE提供；运营全球最大的民航飞行员培训网络，拥有约300台全动飞行模拟器
- **国防与安全**：为美军及盟军提供飞行模拟器、战术训练系统、任务演练系统，客户覆盖美国空军、海军、陆军及多国盟军
- **医疗模拟**：医疗仿真训练

CAE年营收超过40亿加元（2024财年），其中国防与安全业务占比约40%。CAE在军事训练领域的历史可追溯至1960年代，长期为美国及其盟友提供空中、地面和海上平台的训练解决方案。

### CAE USA

CAE USA是CAE Inc.在美国的子公司，总部位于佛罗里达州坦帕，在阿灵顿（弗吉尼亚）、奥兰多（佛罗里达）等地设有运营中心。CAE USA在2023年完成对L3Harris技术公司旗下L3 Link Simulation and Training业务的收购，这项收购使CAE一跃成为美军最大的训练模拟系统独立供应商，并直接承接了SCARS主承包商角色。

## SCARS项目：模拟器通用架构

### 项目概述

SCARS（Simulators Common Architecture Requirements and Standards）是美国空军总部训练局（HAF/A3T）发起的维持性倡议项目，采用IDIQ（不确定交付/不确定数量）合同模式，5年基础期（2020-2025）+ 5个1年选项期（至2030年）。CAE USA担任主承包商，负责领导行业团队为美国空军开发统一模拟器通用架构。

### 项目规模

- **覆盖范围**：50+飞行平台、约2,400台训练设备、300个站点
- **支持司令部**：ACC（空战司令部）、AFGSC（全球打击司令部）、AFRC（预备役）、AFSOC（特种作战）、AMC（空中机动）、PACAF（太平洋空军）、USAFE（欧洲空军）共7个主要司令部（MAJCOM）
- **合同总额**：估计总价值约$700-900M（10年期）

### 三阶段渐进实施

SCARS采用渐进式实施策略：

**第1阶段：高效网络安全（Efficient Cybersecurity）**
- 集中化网络维护：漏洞扫描、补丁、防病毒和企业级RMF控制
- 集中化训练设备扫描/补丁管理，显著减少训练系统维护停机，加速ATO获取
- 首个标准SMP17：远程ACAS扫描

**第2阶段：通用架构（Common Architecture）**
- 训练系统主要功能迁移至通用基础设施
- 支持向云/虚拟计算完全迁移
- 通用建模和标准接口
- 虚拟化非硬件依赖软件

**第3阶段：标准应用（Standard Applications）**
- 训练系统采用通用应用
- 通用模型库和通用应用托管基础设施
- 仅不可虚拟化的软件为训练器特有

### 核心技术架构

SCARS架构包含五个关键组件：

| 组件 | 功能 |
|------|------|
| SOC（安全运营中心） | OPE和SCARS WAN远程管理、中央仪表板、帮助台 |
| OPE（本地设备） | 本地云、可扩展通用安全控制、托管仿真应用 |
| WAN（广域网） | 利用现有伙伴网络，概念验证中 |
| 授权训练系统 | 无ATO不接入SCARS |
| SCARS标准体系 | 包含OPE集成、安全控制、认证流程、数据、DevSecOps、架构共7大标准 |

### 采纳的行业标准

SCARS积极采纳国际开放标准，包括OGC CDB（地理空间数据）、SISO CIGI（图像生成器接口）、IoT（机组控制接口）、ARINC-610C（模拟器功能）、OMG SysML（MBSE建模）、Open XR（扩展现实）等。

## MOSA关联：训练系统开放标准化的意义

### SCARS在MOSA生态中的定位

SCARS是MOSA（模块化开放系统方法）在军事训练模拟领域的系统性实施，在DoD MOSA生态中具有以下关键意义：

1. **规模最大**：覆盖50+平台、2400+设备、7个MAJCOM，是国防部最大规模的MOSA部署项目之一
2. **企业级方法**：不是单个平台的MOSA，而是整个空军训练基础设施的MOSA——从碎片化走向企业级统一
3. **维持性路径**：采用维持性采办路径（非新研），通过增量渐进方式为庞大存量设备建立MOSA
4. **网络安全驱动**：RMF合规是SCARS的首要驱动力，网络安全先行再推进架构标准化

### 在GRA体系中的位置

在[[政府参考架构|政府参考架构（GRA）体系]]中，SCARS被列为**系统级GRA**，在[[开放架构层级]]中处于**第3层（设备级）**，与ABMS GRA（战斗管理）、PNT GRA（导航）同属系统级功能域架构。三军备忘录（2024）中列出的六大已验证MOSA标准中，SCARS是与OMS/FACE/SOSA/WOSA/AMS GRA并列的训练领域标准。

### 与其他MOSA标准的互补

- **SCARS ↔ FACE**：训练系统可复用FACE标准的航电软件组件
- **SCARS ↔ SOSA**：传感器模拟可采用SOSA硬件标准
- **SCARS ↔ OMS/UCI**：SCARS架构标准中引用OMS作为空载器接口标准
- **SCARS ↔ ABMS GRA**：训练系统需模拟ABMS网络环境以支持分布式任务训练

### 行业意义

SCARS的开放标准化对训练行业有深远影响：
- **打破供应商锁定**：政府拥有架构和标准，通过标准化接口促进多家供应商竞争
- **降低全生命周期成本**：通用架构减少独特开发和维护，行业估计可降低30-50%的全生命周期成本
- **加速能力更新**：通过通用模型库和标准接口，新能力可从S&T快速扩散至全部训练设备
- **支持分布式任务训练**：标准化的联网能力使分散在不同站点的模拟器可组成联合训练网络

## 相关项目与客户

### 初始平台

SCARS的首个任务订单覆盖**9个站点**，将新标准整合到**A-10**和**KC-135**平台中。后续将逐步扩展到全部50+平台。

### 其他CAE USA主要项目

除SCARS外，CAE USA还参与或承担多项美军训练项目：

- **美国空军**：C-130、F-16、KC-135、A-10、B-52等平台的飞行模拟器开发和维护
- **美国陆军**：航空训练系统（ATS）、通用传感器训练系统
- **美国海军**：P-8A波塞冬海上巡逻机训练系统
- **美国特种作战司令部**：MC-130J、CV-22倾转旋翼机等特种作战飞机训练系统
- **盟军客户**：北约多国空军模拟训练系统、日本、韩国、澳大利亚等盟友的训练项目

### 合同伙伴

CAE USA在SCARS项目中领导的行业团队包含多家分包商，涵盖网络安全、仿真建模、网络基础设施等专业领域。CAE也通过收购L3 Link Simulation and Training（前身为L3的Link仿真与训练业务，历史可追溯至Edwin Link于1929年创立的Link Aviation Devices）继承了其与空军长期合作的训练合同关系。

## 战略影响

CAE USA对SCARS项目的主导地位反映了几个重要趋势：

1. **训练领域的行业整合**：CAE通过收购L3 Link业务，从民航模拟器龙头转变为美军最大训练系统独立供应商，集中度提升趋势明显
2. **MOSA从政策到实践**：SCARS作为覆盖范围最广的MOSA实施之一，为其他领域的开放架构推广提供了可参考的模板
3. **网络安全成为架构变革首要驱动力**：训练设备网络安全合规的迫切需求推动了从碎片化到标准化的架构变革
