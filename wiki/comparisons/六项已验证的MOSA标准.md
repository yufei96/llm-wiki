# 三军已验证开放标准生态（6大标准）

## 概述

2024年12月三军联合备忘录首次在最高政策层面明确列举了各军种已成功开发、演示和验证的6大开放标准。这是MOSA标准生态的权威清单。

## 6大已验证标准

| 标准 | 全称 | 领域 | 主导方 |
|------|------|------|--------|
| **OMS/UCI** | Open Systems/Universal Command and Control Interface | 通用指挥控制接口 | 跨军种 |
| **SOSA** | Sensor Open Systems Architecture | 传感器系统架构 | The Open Group |
| **AMS GRA** | Agile Mission Suite Government Reference Architecture | 敏捷任务套件 | 政府 |
| **WOSA** | Weapons Open Systems Architecture | 武器系统架构 | 跨军种（超音速武器对齐） |
| **FACE** | Future Airborne Capability Environment | 航空电子软件 | The Open Group |
| **VICTORY** | Vehicular Integration for C4ISR/EW Interoperability | 车辆C4ISR互操作 | 行业联盟 |

## 标准时序与关系

```
2010: FACE 1.0发布
2012: VICTORY启动
2013: OMS/UCI标准化
2017: SOSA联盟成立
2019: CMOSS/MORA整合
2021: WOSA用于超音速武器
2024: 三军备忘录统一确认6大标准
```

## 关键洞察

- **AMS GRA 是新面孔**：此前 wiki 中未出现，三军备忘录首次在政策层面确认
- **WOSA 从工具→标准**：现有 wiki 将 WOSA 描述为架构评估工具，但三军备忘录将其定义为武器系统接口标准（超音速武器等场景对齐）
- **标准生态趋同加速**：6个标准覆盖了从软件（FACE）到硬件（SOSA）到武器（WOSA）到车辆（VICTORY）的完整栈
