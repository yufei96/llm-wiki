# DJI开发者平台：SDK体系概览

## 摘要

DJI开发者平台提供覆盖移动端、机载端、负载端、云端的6大SDK：Edge SDK、Mobile SDK、Payload SDK、UX SDK、Windows SDK、Onboard SDK。通过抽象产品类和组件类实现高度可扩展的架构设计。

## 关键信息

- **平台位置**：https://developer.dji.com/cn/
- **SDK数量**：6大SDK
- **支持产品**：覆盖Mavic、Matrice、Mini、Phantom等全系列
- **开发平台**：iOS/Android（Mobile/UX）、Linux/RTOS（Payload）、Windows（Windows SDK）、机载计算机（Onboard）、云端（Edge/Cloud API）

## 主要SDK

| SDK | 定位 | 支持平台 |
|-----|------|----------|
| Mobile SDK | 移动端应用控制无人机 | iOS/Android |
| Payload SDK | 负载设备开发 | Linux/RTOS |
| Onboard SDK | 机载计算机应用 | 机载计算机 |
| Edge SDK | 无人机机场/云端管理 | DJI Dock系列 |
| UX SDK | 用户界面快速构建 | iOS/Android |
| Windows SDK | Windows平台开发 | Windows |

## 架构特点

- 抽象产品类和组件类，同一套代码控制不同产品
- 新产品发布时现有应用即可兼容（需更新SDK版本）
- 组件化设计：飞行控制器、云台、相机、遥控器、无线链路独立抽象
