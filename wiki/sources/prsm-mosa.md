# 精确打击导弹（PrSM）MOSA实施案例

## 摘要

精确打击导弹（PrSM）是美国陆军下一代远程精确打击导弹，由洛克希德·马丁研制，旨在替代ATACMS战术导弹系统。PrSM采用模块化开放系统架构设计，支持通过增量（Increment）方式逐步扩展能力。Increment 1为基础版本，Increment 2增加多模导引头具备打击移动海上目标能力，Increment 3计划扩展射程。GAO-25-106931确认PrSM为20个受审项目中14个实施MOSA的项目之一。

## 关键要点

- **MOSA实施状态**：实施中，被GAO-25-106931确认为MOSA项目
- **模块化增量设计**：支持Increment 1-4逐步扩展能力，无需重新设计整枚导弹
- **平台类型**：全新设计（Greenfield），替代ATACMS
- **采办路径**：主要能力采办（MDAP），采购类别IB
- **合同类型**：成本加激励费用（CPIF）
- **竞争效果**：Increment 1竞争在洛克希德·马丁和雷神之间进行，洛克希德胜出
- **产能提升**：2026年签署7年框架协议，年产能提升至550枚
- **无成本效益分析**：与GAO审查的其他项目一样，未进行正式MOSA成本效益分析

## 模块化子系统

1. 多模导引头（Increment 2新增，支持移动目标跟踪）
2. 战斗部模块（可根据任务需求更换）
3. 推进系统（固体火箭发动机）
4. 制导与导航系统（GPS/INS，可扩展）
5. 数据链模块（支持飞行中目标更新）
6. 发射适配模块（兼容M142 HIMARS和M270 MLRS）

## 主要里程碑

| 时间 | 事件 |
|------|------|
| 2016 | 项目启动，进入TMRR阶段 |
| 2019 | 洛克希德·马丁获得EMD合同 |
| 2021 | 完成关键设计评审（CDR） |
| 2023 | Increment 1首次成功飞行测试 |
| 2024 | 进入低速初始生产（LRIP）阶段 |
| 2026-03 | Increment 2完成首次飞行测试（350公里射程） |
| 2026 | 签署7年生产框架协议 |
