# DFARS Case 2021-D005: 模块化开放系统方法合同条款

## 摘要

DFARS 2021-D005是国防部针对[[MOSA与国防采办|MOSA]]合同条款的法规修订案，旨在实施三项NDAA（FY2012 Sec 815、FY2017 Sec 809、FY2021 Sec 804）的MOSA合同要求。该修订适用于**全军种**，属于统一采购法规层面的变更，而非特定采购项目。

**当前状态**：进行中——预通知拟议规则制定（ANPRM）已于2023年11月17日发布，最终规则尚未发布。

**核心变更方向**：
1. 新增MOSA、IID（接口实现数据）、IIS（接口实现软件）等术语定义
2. 修订技术数据和软件许可权利条款，使政府获得接口实现数据/软件的政府目的权利（GPR）
3. 要求采办计划中识别模块化系统接口及相关技术数据/软件

该修订同时适用于新系统和现有系统（[[MOSA棕地与绿地对比|Brownfield vs Greenfield]]），对所有采办路径均有约束力。

## 关键条款

### 术语定义
- **模块化系统接口（Modular System Interfaces）**：系统模块间定义明确的接口边界
- **接口实现数据（IID, Interface Implementation Data）**：描述接口实施步骤、序列、特性和条件的数据
- **接口实现软件（IIS, Interface Implementation Software）**：支持接口功能实现的软件组件
- **形式、适配和功能数据/软件（Form, Fit, and Function）**：接口规范的形式、适配和功能子集

### 数据权利变更
本法规的核心是[[供应商锁定|打破供应商锁定]]的IP权利机制：
- **政府目的权利（GPR）**：政府获得模块化系统接口IID/IIS的政府目的权利，即使由承包商专有出资开发
- **受限权利修订**：允许在必要时向政府以外方披露IID/IIS以支持拆分/重新集成活动
- **无限权利**：政府对形式、适配和功能数据/软件拥有无限权利

### 法律依据
- 10 U.S.C. 4401（[[美国法典第10编4401-4403 MOSA法|MOSA法定要求]]）
- 10 U.S.C. 3771
- DoDI 5010.44
- FY2012 NDAA Sec 815 / FY2017 NDAA Sec 809 / FY2021 NDAA Sec 804

## 对MOSA实施的影响

### 积极影响
1. **促进竞争**：通过要求在合同中识别模块化系统接口并授予政府GPR，打破供应商锁定，促进子系统层面的竞争采购
2. **标准化合同语言**：为MOSA实施提供统一的合同条款框架，减少各项目自行定义的差异
3. **数据权利保障**：确保政府拥有接口数据/软件权利，支持系统的模块化拆分和重新集成

### 实施挑战
1. **GPR授予条件复杂**：政府目的权利的授予条件需要逐案评估
2. **现有合同修改**：修订同时适用于新系统和现有系统合同修改，适用性需逐案评估
3. **行业担忧**：合规负担和知识产权披露风险
4. **交叉影响**：与伴生案例DFARS 2019-D044需综合考量
5. **不确定因素**：行业反馈内容、最终规则发布时间、盟友互操作性影响

### 与[[MOSA五支柱|五大支柱]]的关系
DFARS 2021-D005直接支撑五大支柱中的**指定接口**和**开放标准**支柱，通过合同法律层面强制要求接口开放和数据权利共享。
