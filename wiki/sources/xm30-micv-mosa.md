# XM-30机械化步兵战车MOSA设计案例

## 摘要

XM-30机械化步兵战车（原OMFV）是美国陆军下一代步兵战车项目，旨在替代M-2布雷德利。XM-30是陆军首个从设计之初就全面采用模块化开放系统架构（MOSA）和数字工程的地面战斗车辆。项目采用中间层采办快速原型（MTA-RP）路径，目前由通用动力陆地系统公司（GDLS）和美国莱茵金属车辆公司两家竞争承包商进行原型开发。

## 关键要点

- **MOSA实施状态**：从项目启动即嵌入MOSA要求，是陆军首个全面MOSA地面战斗车辆
- **陆军自有标准**：采用Army-defined and Army-owned开放标准，而非直接采用SOSA/CMOSS/FACE等商业标准体系
- **平台类型**：全新设计（Greenfield），替代M-2布雷德利
- **采办路径**：中间层采办快速原型（MTA-RP）
- **竞争结构**：Phase 2有5家公司参与，Phase 3&4缩小至2家（GDLS和American Rheinmetall）
- **数字工程**：首个从概念设计起全面使用MBSE的地面战斗车辆，数字孪生贯穿设计、仿真、测试全过程
- **可选有人驾驶**：模块化架构支撑"可选有人驾驶"核心能力
- **里程碑B争议**：2026年2月陆军领导层暂停签署最终里程碑B文件以保持竞争开放

## 主要里程碑

| 时间 | 事件 |
|------|------|
| 2021-07 | Phase 2合同授予（5家公司，共约2.994亿美元） |
| 2023-06 | Phase 3&4合同授予（2家公司，共约16亿美元） |
| 2024-08 | 初步设计评审（PDR）完成 |
| 2025-06 | 里程碑B批准 |
| 2026-02 | 里程碑B暂停签署以保持竞争 |
| 2026秋 | 第1骑兵师接收原型车进行作战测试 |
| FY2028 Q1 | 目标里程碑C |
| FY2029 | 首批车辆列装 |

## 项目规模

- Phase 2约2.994亿美元 + Phase 3&4约16亿美元
- FY2027预算5.47亿美元用于19辆原型车采购
- 全速生产阶段总价值预计数百亿美元

## 关键模块化子系统

1. 任务计算系统（开放式计算架构）
2. 传感器套件（光电/红外、雷达）
3. 主动防护系统（APS）
4. 武器站（模块化配置）
5. 通信系统（战术数据链）
6. 混合动力驱动模块
