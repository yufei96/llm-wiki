# DFARS第227部分：专利、数据及版权

## 概述

DFARS第227部分是国防联邦采购条例补充中关于**知识产权**的核心章节，覆盖政府合同中的专利权、数据权和版权。实施的法律依据包括10 U.S.C. 3013、3208(d)、3771-3775、3781-3786等。

DoD活动应使用Subpart 227.71和227.72代替FAR Subpart 27.4的指导（227.400）。

## 主要子部分结构

| 子部分 | 内容 | MOSA相关度 |
|--------|------|-----------|
| 227.3 | 政府合同下的专利权 | 低 |
| 227.4 | 数据和版权（范围声明，实际使用227.71/72） | — |
| 227.6 | 外国许可和技术援助协议 | 低 |
| 227.70 | 侵权索赔、许可和转让 | 低 |
| **227.71** | **技术数据及相关权利** | **核心** |
| **227.72** | **计算机软件、文档及相关权利** | **核心** |

## Subpart 227.71 — 技术数据及相关权利（MOSA核心）

### 政策（227.7103-1）

DoD政策是仅获取满足机构需求所需的**技术数据及其中的权利**。具体要求：

- 征询书和合同应**指定**要交付的技术数据及交付时间表
- 建立可交付数据项的**独立合同行项目**，要求单独定价
- 要求报价方识别受限制的技术数据
- 不得以报价条件或授标条件要求报价方放弃私资开发项的数据权（227.7103-1(c)）
- 不得因政府使用私资开发数据的权利受限而**阻止或劝阻**承包商提供此类项目

### 技术数据获取（227.7103-2）

合同官应与数据管理人员密切合作。数据需求需权衡：
- 承包商在私资开发项目中的经济利益
- 政府获取、维护、存储、检索和保护数据的成本
- 再采购需求
- 维修维护和大修方案
- 备件考虑

当预期政府将获得无限或政府目的权利用于竞争性备件采购时，此类数据应识别为可交付数据项。

### 三类标准许可权利（227.7103-4 & 227.7103-5）

政府通过不可撤销许可获得技术数据权利。权利范围取决于**资金来源**：

#### 无限权利（Unlimited Rights）

政府在以下情况获得无限权利：

| 编号 | 条件 |
|------|------|
| (1) | 项目由政府资金**全额**开发 |
| (2) | 合同执行中产生的研究、分析、测试数据（作为绩效要素指定） |
| (3) | 政府资金全额创建，且不要求开发项目的数据 |
| (4) | Form, fit, and function数据 |
| (5) | 安装、操作、维护或培训所必需的数据（非详细制造/工艺数据） |
| (6) | 对政府提供数据的修正或更改 |
| (7) | 已公开可用或已被承包商无限制释放的数据 |
| (8) | 在其他政府合同下已获得无限权利的数据 |
| (9) | 政府目的/有限权利且限制条件已过期的数据 |

#### 政府目的权利（Government Purpose Rights）

- 适用于**混合资金**开发的项目相关数据
- 名义有效期**5年**，可协商调整
- 期满后政府获得无限权利
- 有效期内政府不得将数据用于商业目的
- 释放给第三方前需签署使用和保密协议（227.7103-7）

#### 有限权利（Limited Rights）

- 适用于**全额私资**开发的项目相关数据
- 政府不得在未经承包商许可的情况下在政府外使用、释放或披露
- 例外：紧急维修、政府支持承包商、特定外国政府评估用途
- 断言有限权利者须在释放前收到通知

#### 专门协商许可（Specifically Negotiated License）

- 当标准权利不满足政府需求或政府愿接受较少权利以换取其他对价时可协商
- 承包商无义务提供超出标准的权利，合同官无义务接受低于标准的权利

### 合同条款（227.7103-6）

| 条款 | 用途 |
|------|------|
| **252.227-7013** | 非商用产品/服务的技术数据权利（核心条款） |
| **252.227-7015** | 商用产品/服务的技术数据权利 |
| **252.227-7025** | 限制性标记政府提供信息的使用/披露限制 |
| **252.227-7026** | 技术数据/软件延期交付 |
| **252.227-7027** | 技术数据/软件延期订购 |
| **252.227-7028** | 先前已交付的技术数据/软件 |
| **252.227-7030** | 技术数据—付款扣留 |
| **252.227-7037** | 限制断言验证 |

### 分包商权利（227.7103-15）

- 10 U.S.C. 3771为**所有层级**分包商提供与主承包商相同的数据权保护
- 10 U.S.C. 3782允许分包商直接与政府处理权利验证事务
- 主承包商须将252.227-7013、7025、7028条款无修改地纳入分包合同

### 延期交付和延期订购（227.7103-8）

| 机制 | 条款 | 期限 | 用途 |
|------|------|------|------|
| **延期交付** | 252.227-7026 | 项目验收后2年 | 推迟数据交付时间 |
| **延期订购** | 252.227-7027 | 项目验收后3年 | 合同签订时未确定但有潜在需求的数据 |

## Subpart 227.72 — 计算机软件及相关权利

与227.71结构平行，适用于计算机软件和软件文档。权利类别相同（无限/政府目的/有限），关键条款为252.227-7014（非商用软件）。

## 与MOSA的关系

本部分是MOSA合同执行的**法律基础设施**：

- MOSA"政府拥有接口定义"原则的法律依据来自227.71的无限权利条款
- 252.227-7013是[[航空PEO-MOSA实施指南]]中SOW/PWS检查清单的核心条款
- 分包商权利保护（227.7103-15）确保模块化架构中各层级供应商的权利一致性
- 延期订购机制（227.7103-8）支持MOSA的渐进式数据需求确定
