# UAI — 通用武器接口标准

## 概述

- **全称**：Universal Armament Interface
- **定位**：标准化机载平台与武器之间的接口——使任何平台可携带任何 UAI 兼容武器

## 核心价值

UAI 解决的问题：传统上每种武器有自己的专属接口，更换武器类型需要修改平台软硬件。UAI 将武器接口标准化为：

1. **物理接口**：标准化的电气和机械连接
2. **逻辑接口**：标准化的消息协议（基于 OMS/UCI）
3. **安全分离**：标准化的安全释放机制

## 在 MOSA 生态中的位置

UAI 是 **WOSA 在机载武器领域的具体实现**：

| 标准 | 领域 | 接口类型 |
|------|------|---------|
| OMS/UCI | 任务级消息 | 平台↔任务系统 |
| FACE | 航电软件 | 应用↔平台 |
| **UAI** | **武器集成** | **平台↔武器** |
| WOSA | 武器架构 | 武器内部模块化 |

## 采办意义

UAI 的标准化直接降低武器集成成本和周期——新武器只需通过 UAI 认证即可在任何 UAI 兼容平台上使用，无需重新集成和飞行测试。
