# 美国陆军采办项目组合 2024

## 概述

美国陆军采办、后勤与技术助理部长（ASA(ALT)）发布的2024年Q3版采办项目组合概览。这是陆军最全面的**年度武器系统目录**，涵盖陆军所有在研和现役采办项目，按功能域分14个组合。文件同时包含ASA(ALT)组织架构、科学技术（S&T）投资概况和PEO/合作伙伴组织介绍。

## 核心内容

### 组织结构

- **ASA(ALT)**：陆军采办最高领导机构
- **下属DASA**（副助理部长）：分管不同采办领域
- **DASM**：采办与系统管理副主任
- **PEO**（项目执行办公室）：负责具体武器系统项目群
- **合作伙伴组织**：涉及测试、后勤、合同等

### 14个采办项目组合

| 组合 | 代表项目 |
|------|---------|
| **防空反导 (AMD)** | IBCS、LTAMDS、Patriot PAC-3、IFPC-HEL/HPM、M-SHORAD、Sentinel A3/A4、SGT STOUT |
| **弹药** | 大口径/中口径/小口径弹药、炮兵弹药、引信与推进剂、C-DAEM |
| **保障机动** | HMEE-IV、军用桥梁、扫雷机器人 M160、S-MET Inc II |
| **航空** | AH-64E、UH/HH-60M、CH-47F Block II、FLRAA、FTUAS、HADES、ITEP T901、JAGM、Launched Effects |
| **火力** | PrSM Inc 1-4、LRPF、增程加农炮（ERCA）、HIMARS |
| **情报** | 电子战、信号情报、侦察系统 |
| **机动** | M10 Booker、AMPV、JLTV、可选有人战车（OMFV → XM30） |
| **任务指挥** | 指挥所、网络通信、战场管理系统 |
| **防护** | 主动防护系统（APS）、CBRN防护 |
| **机器人与AI** | 机器人战车（RCV）、AI赋能系统 |
| **士兵** | NGSW、IVAS、士兵增强系统 |
| **保障** | 后勤信息系统、供应链现代化 |
| **合成训练环境 (STE)** | 虚拟/构造训练系统 |
| **运输** | 战术车辆、重型运输 |

### S&T 投资概览

- 陆军S&T企业架构
- 五大投资领域：地面、航空、网络C3I、士兵杀伤力、武器系统
- 小企业创新研究（SBIR）和xTech项目
- 技术成熟化倡议（TMI）
- 制造技术（ManTech）
- 基础研究

## 与MOSA的关联

- 多个项目为MOSA案例：FLRAA、XM30 MICV、PrSM
- LTAMDS采用开放架构雷达，与IBCS集成体现模块化防空理念
- 陆军S&T明确支持开放标准和技术成熟化

## 关键数据

- 编制版本：Q3FY24（2024年第三季度）
- 项目组合数：14个功能域
- 覆盖项目：>150个（含弹药子类）
- 涵盖生命周期：从S&T基础研究到列装在役
