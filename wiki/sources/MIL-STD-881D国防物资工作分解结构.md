# MIL-STD-881D — 国防装备工作分解结构标准

## 概述

- **全称**：Work Breakdown Structures for Defense Materiel Items
- **发布方**：DoD
- **定位**：定义国防采办项目的**工作分解结构（WBS）**标准

## 用途

MIL-STD-881D 提供标准化的 WBS 框架，用于：

1. **成本估算**：按标准 WBS 元素分解项目成本
2. **进度管理**：WBS 元素与甘特图和里程碑对齐
3. **合同管理**：合同工作说明按 WBS 组织
4. **跨项目比较**：标准 WBS 使不同项目可比较

## 与 MOSA 的关系

MIL-STD-881D 对 MOSA 实施有间接但重要的影响：

- WBS 元素定义决定了**如何核算模块化成本**
- 标准 WBS 为每个模块/接口**分配独立工作包**提供框架
- 2020 年国防部推进"数据驱动 WBS"，与数字工程（DoDI 5000.97）对齐
