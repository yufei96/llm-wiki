# NAVAIR OSMP Implementation Description Guide v1.0 2025

## 摘要

美国海军航空系统司令部（NAVAIR）2025年发布的开放系统管理计划（OSMP）实施描述指南v1.0。这本指南是海军对DoD MOSA政策的实操级响应，提供了从采办策略到合同条款的完整OSMP编写框架和模板。是理解海军如何将MOSA落实到具体采办项目中的关键文献，2.4MB原始PDF，1409行转换文本。

## 关键信息

- 明确定义OSMP在采办策略中的角色
- 包含CDRL模板、SOW条款建议
- 对接FACE、OMS、WOSA等海军相关标准
- 是osmp-implementation-guide topic页的直接来源
