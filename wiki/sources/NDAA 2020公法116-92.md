# FY2020 国防授权法案（NDAA 2020）— 公法 116-92

## 概述

国家防御授权法案 2020 财年（National Defense Authorization Act for Fiscal Year 2020），公法编号 PL 116-92，2019年12月20日由总统签署生效。该法案是第116届国会通过的首部完整年度国防授权法案，授权 2020 财年国防部军事活动、军事建设和能源部国防活动的拨款。

## 法案结构

法案分为六个部分（Divisions）：

| 部分 | 内容 |
|------|------|
| Division A | 国防部授权（采购、RDT&E、运维、人事、薪酬、医疗、采办政策等） |
| Division B | 军事建设授权 |
| Division C | 能源部国家安全授权及其他授权 |
| Division D | 拨款表 |
| Division E | 2018-2020 财年情报授权 |
| Division F | 其他事项 |

## 核心技术/采办相关条款

- **Section 840**（MOSA实施指南）：要求各军种部长发布模块化开放系统方法（MOSA）实施指南，修改 10 USC §2446b/c
- **Section 231**（数字工程）：要求建立数字工程能力以自动化测试与评估
- **Section 255**（软件战略）：要求国防部范围软件科技战略
- **Section 224**（微电子信任）：要求国防微电子产品满足可信供应链和操作安全标准
- **Section 226**（5G）：在内华达测试训练靶场建立安全5G网络基础设施
- **Section 164**（敏捷DevOps）：要求为 F-35 ALIS 建立敏捷 DevOps 替代方案
- **Section 230**（数字人才）：制定数字专业知识和软件专业人员的人才管理政策

## 军事人员政策亮点

- 扩大性侵预防与应对措施（Subtitle D, Title V）——加强军事司法改革
- 多元化与包容性战略计划（Sec. 529）
- 延长预备役自杀预防与韧性项目
- PFAS 相关：禁止在 MRE 食品包装中使用 PFAS（Sec. 329），要求以无氟灭火剂替代 AFFF（Sec. 322-324）

## 笔记

PL 116-92 的时代背景：在中美战略竞争加剧和国防部推进「国家防御战略」的背景下通过。法案体现了国会两方面的关切：一是通过 MOSA、数字工程、5G、微电子等条款推动国防采办体系现代化；二是通过性侵防治、多元化、PFAS 环保等条款回应社会和军队内部治理问题。Section 840 的 MOSA 条款是 MOSA 法律框架从「仅 MDAP」走向「全项目覆盖」的重要中间步骤。
