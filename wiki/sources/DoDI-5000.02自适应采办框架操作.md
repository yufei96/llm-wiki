# DoDI 5000.02 摘要（2020年，2022年变更1）

## 文档元数据

- **生效日期**：2020年1月23日；变更1：2022年6月8日
- **发布机构**：USD(A&S) — 国防部采办与维持副部长
- **替代**：2015年1月7日版DoDI 5000.02（已取消并重新编号为5000.02T，现也已取消）

## AAF — 自适应采办框架

**目的**：通过**6条路径**及时向最终用户交付有效、适用、可生存、可持续和经济可承受的解决方案。

## 6条采办路径

### 1. 紧急能力采办
- **目标**：**2年内**部署能力
- **成本上限**：5.25亿美元研发或30.65亿美元采购（FY2020不变价美元）
- **时间线**：数周规划，数月开发/生产
- **参考**：DoDI 5000.81

### 2. 中间采办层级（MTA）
- **目的**：快速原型（**5年内**部署）或快速列装（**6个月内**开始生产，**5年内**完成）
- **豁免**：不受JCIDS或DoDD 5000.01程序约束
- **注意**：如项目超过MDAP阈值需USD(A&S)批准
- **参考**：DoDI 5000.80

### 3. 重大能力采办
- **目的**：持久的军事独特能力
- **结构**：分析 → 设计 → 开发 → 集成 → 测试 → 评估 → 生产 → 保障
- **MDAP级别**：USD(A&S)担任ACAT ID项目的里程碑决策机构
- **参考**：DoDI 5000.85

### 4. 软件采办
- **目的**：敏捷软件开发
- **参考**：DoDI 5000.87

### 5. 国防业务系统（DBS）采办
- **参考**：DoDI 5000.84

### 6. 服务采办
- **参考**：DoDI 5000.74

## 决策层级

| 角色 | 职责 |
|------|------|
| **MDA/DA**（里程碑决策机构） | 项目决策机构；指定决策点和程序 |
| **PEO**（项目执行官） | 组合风险、成本、进度、性能、互操作性 |
| **PM**（项目经理） | 规划、准备、执行采办和产品支持策略 |
| **产品支持经理** | 在PM指导下制定综合产品支持策略 |

## 关键职责

- **USD(A&S)**：担任ACAT ID项目的MDA（物质发展决策、里程碑A/B/C）
- **USD(R&E)**：对ACAT ID进行独立技术风险评估（ITRA）；确认技术可行性
- **CAE**（组件采办执行官）：执行指示，可提交豁免申请

## 通用程序（4.1节）

- **裁剪**：项目经理识别并推荐监管信息供MDA批准——根据项目定制文档
- **多路径**：项目经理可组合路径；必须定义转换点并协调进入要求
- **赛博安全**：必须在项目规划早期和持续解决
- **数据权利**：考虑采购数据可交付物和竞争性采办的许可权利
- **产品支持与经济可承受性**：在早期项目规划中优先考虑
- **风险管理**：PM必须建立项目；确定哪些风险需缓解、哪些可接受

## 对MOSA研究的启示

论文（Shah 2021）引用DoDI 5000.02作为MOSA授权来源。但实际的MOSA指令在**更好采购力3.0**和**USC 10 §2446a**中。DoDI 5000.02是项目必须遵循的**流程框架**（AAF），而非MOSA政策本身。2015年版本可能引用了MOSA，但2020年版本是重组后的框架文档。**MOSA具体实施指导可能在DoDI 5000.85（重大能力采办）或单独的USD(A&S)备忘录中。**
