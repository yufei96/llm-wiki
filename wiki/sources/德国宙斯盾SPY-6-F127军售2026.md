# 德国Aegis/SPY-6 F127护卫舰军售案（2026年4月）

## 概述

2026年4月，美国国务院批准向德国出售Aegis/SPY-6综合战斗系统，用于装备德国海军F127下一代护卫舰。该军售案估计总成本119亿美元，涵盖8套完整舰载战斗系统。这标志着德国海军从欧洲Tacticos/APAR系统向美制Aegis/SPY-6系统的重大战略转向，也是SPY-6雷达在欧洲的首个应用案例。

## 核心内容

### 军售规模

| 参数 | 数值 |
|------|------|
| 估计总成本 | 119亿美元 |
| 舰载系统套数 | 8套完整系统 |
| 主承包商 | Lockheed Martin, RTX Corporation |
| 目标服役时间 | 2030年代中期 |

### 每套舰载系统核心装备

- **Aegis综合战斗系统（ICS）MK 6 MOD X**：最新版本宙斯盾作战系统
- **AN/SPY-6(V)1 AESA S波段雷达**：美国海军最新一代舰载防空反导雷达
- **MK 41 Baseline VIII垂直发射系统**：96单元配置
- **协同交战能力（CEC）**：实现跨平台传感器-射手网络化协同
- **AN/SPQ-9B雷达**：X波段水面搜索与火控雷达
- **AN/WSN-12惯性导航系统**

### 附加装备

- 9套指挥控制处理器
- 10套MIDS舰载现代化系统
- 9座MK 45舰炮
- 3套AN/SLQ-32(V)6电子战系统
- MK 38 MOD 4反无人机武器系统

### 战略背景与意义

- **项目演变**：F127原计划建造6艘替换F124萨克森级护卫舰，因欧洲安全环境变化扩展至8艘
- **能力转型**：德国海军从传统防空转向弹道导弹防御（BMD）、综合防空反导和对地攻击
- **设计基础**：基于MEKO A400 AMD概念，满载排水量预计超过12,000吨
- **供应链多元化**：SPY-6雷达部分组件由日本三菱电机（MELCO）生产，体现Aegis供应链的全球化布局
- **欧洲转向**：德国从长期使用的Thales APAR系统转向美制宙斯盾，反映了欧洲安全架构的深层变化

## 笔记

德国F127军售案是SPY-6雷达系列国际化的里程碑。此前SPY-6主要装备美国海军Flight III阿利·伯克级驱逐舰和星座级护卫舰。德国选择Aegis/SPY-6而非欧洲本土系统，关键在于SPY-6的弹道导弹防御能力和与北约盟军（尤其是美国海军）的互操作性。96单元MK 41 VLS配置使F127的弹药容量远超传统欧洲护卫舰，体现了德国海军从"护卫"到"主力防空战舰"的角色转变。该军售还需国会批准且最终金额可能因谈判调整。
