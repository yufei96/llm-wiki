# 克里斯·加勒特 — USAF数字化转型演讲（2024年9月）

## 概述

- **演讲者**: Chris Garrett, AFLCMC/EN-EZ，空军生命周期管理中心系统工程高级负责人
- **主题**: "USAF数字化转型: Think Big, Start Small, Scale Fast"
- **组织**: 空军装备司令部(AFMC)
- **分发声明**: Distribution Statement A — 批准公开发布，无限制

## 战略背景：能力获取速度劣势

演讲以中美海军能力列装趋势对比开场:

- **中国**: DF-21D反舰弹道导弹、J-15舰载战斗机、协同交战能力、高超音速武器、YJ-12/18反舰巡航导弹、LUYANG III驱逐舰、反卫星能力、Type 055巡洋舰——多项能力从初始估计到实际列装时间远短于美国同类项目
- **美国**: 海军综合火控-防空(NIFC-CA)、F-35联合攻击战斗机、SM-6标准导弹、高空反潜武器、下一代干扰机、防空反导雷达、CG(X)巡洋舰、海上打击战斧——普遍延迟
- **根本原因**: 中国"获取设计——无DoD 5000——无官僚程序要求"。美国作战优势相对PLA(N)持续被侵蚀。

## AFMC数字化战略：六大倡议

1. **结构化和安全化数据** — 低摩擦的跨组织协作和决策
2. **促进AFMC范围内数字培训** — 为完全数字生态系统协作做准备
3. **促进DMM工具获取** — 为数字作战配备动态工具箱
4. **制定各AFMC组织的数字战略** — 摒弃陈旧实践，为敏捷采办和保障铺路
5. **灌输数字优先文化** — 在持续变化的威胁环境中革新AFMC的业务方式
6. **现代化IT基础设施** — 持续赋能快速企业解决方案

## 笔记

1. **战略紧迫性**: Garrett选择以中美海军能力对比开场，将数字化转型从"技术优化"提升为"生存问题"——这不只是工具升级，而是对竞争对手速度优势的制度性回应。
2. **GRA的核心定位**: 政府参考架构不是建议性的设计参考，而是合同工具——政府"拥有架构并在现代工具中建模"，用于持续审查供应商设计，根本上改变了采办方与供应商的权力关系。
3. **DMM中的"Think Big, Start Small, Scale Fast"**: 三个短语各对应一个操作——"Think Big"=GRA体系的全域愿景，"Start Small"=AV GRA从79个电气文档起步，"Scale Fast"=扩展到所有功能域和40+平台。
4. **数字化≠数字化现有流程**: "数字化新流程"的区分是关键——将纸质PDF转为模型是数字化，但重新设计跨生命周期的标准化模型化流程才是真正的数字化转型。
5. **OMS作为"隔离器"**: 在目标架构中，OMS被明确标注为隔离器角色——将使命功能（软件服务）与平台特定实现解耦，是实现MOSA中模块独立竞争的核心机制。
6. **与克里斯-加勒特另一身份关联**: Garrett同时是克里斯-加勒特公司的创始人（参见entities/克里斯-加勒特.md），该公司为政府GRA建设提供工具和服务。
