# NAVAIR MOSA官方页面

## 概述

NAVAIR（海军航空系统司令部）MOSA官方页面是海军在模块化开放系统方法（[[MOSAIC方法]]）方面最权威的公开信息来源。该页面由NAVAIR PMA-209（空战电子办公室）维护，全面介绍了海军MOSA战略、法规要求、开放标准生态和三军互操作性演示成果。

## 核心内容

### MOSA定义与战略价值

MOSA是经国防部批准的集成业务和技术策略，旨在通过实施开放系统/开放架构标准，在系统全寿命周期内实现竞争性和可负担的采办与保障。其核心要素包括：

- **模块化设计**：松耦合、高内聚的系统结构
- **开放接口标准**：基于广泛支持的共识标准
- **跨平台复用**：避免为每个平台开发重复功能方案
- **技术与合同融合**：将技术要求与采购机制和法律考量相结合

### NDAA 2021第804条 — MOSA立法强制要求

2021年《国防授权法案》第804条大幅扩展了MOSA实施要求：
- 要求国防部发布法规和指南，促进对模块化系统接口的访问和使用
- 每个武器系统的项目官员必须表征武器系统的期望模块化程度
- 采办合同必须包含模块化系统接口交付要求
- 适用范围扩展到主要国防采办项目之外的所有项目（[[国防部指令5000.85-2020]]）
- 修订10 U.S.C. § 2320，授予政府对私资或部分联邦资金开发的模块化系统接口的目的权
- 要求国防部建立MOSA接口中央存储库

### MOSA实施指南

2025年2月发布的MOSA实施指南（MOSA Implementation Guidebook）面向海军和海军陆战队所有项目经理，是MOSA落地操作的核心参考文件。三军部长联合备忘录（2024年12月签署）进一步巩固了MOSA作为作战必要条件的地位。

### 海军MOSA项目增长

- 2020年：2个MOSA试验项目
- 2021年：5个MOSA项目
- 当前：越来越多的NAVAIR项目办公室对MOSA产生兴趣
- PMA-209航空电子架构团队（AAT）与陆军和空军OA专家协作

### 三军开放架构互操作性演示（TSOA-ID）

NAVAIR PMA-209与陆军PEO Aviation、DEVCOM C5ISR中心、AFLCMC以及行业伙伴和学术界合作，成功展示了三军技术平台上的开放标准集成和使用。关键成果：
- 陆军通信卡安装并集成在各军种底盘上，实现跨演示场地通信
- 使用HOST/SOSA硬件和FACE软件实现通信卡的快速集成
- **集成时间从数月/数年缩短到仅数周**

### 关键开放标准

NAVAIR PMA-209积极支持以下开放标准：

| 标准 | 全称 | 领域 |
|------|------|------|
| **HOST** | Hardware Open Systems Technologies | 高性能嵌入式计算硬件 |
| **FACE™** | Future Airborne Capability Environment | 航空电子软件 |
| **SOSA™** | Sensor Open System Architecture | C4ISR传感器与子系统 |
| **OMS** | Open Mission Systems | 软件服务与硬件子系统接口 |

### 领导层表态

- **VADM Dean Peters**（NAVAIR司令，已退役）：强调开放标准的关键优势——跨平台系统可移植性、快速能力插入、改进互操作性、降低成本
- **BG Rob Barrie**（陆军航空PEO）：承诺将MOSA纳入源选择标准，不合规的项目将被淘汰
