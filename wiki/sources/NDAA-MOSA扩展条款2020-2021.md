# 2020/2021财年国防授权法案MOSA扩展条款

## 摘要

2020 NDAA（PL 116-92，2019年12月20日签署）Section 840和2021 NDAA（PL 116-283，2020年12月20日签署）Section 804是MOSA法定要求演进的关键里程碑。2020 NDAA要求各军种部长发布MOSA实施指南，将首席系统工程师（LSE）职责纳入MOSA实施。2021 NDAA将MOSA要求从重大国防采办项目（MDAP）扩展至所有采购项目，并要求建立MOSA使能的接口存储库。两部法律的MOSA条款已编入美国法典第10编第4401节（10 USC 4401）。

## 关键要点

- **2020 NDAA Section 840核心**：要求军种部长发布MOSA实施指南；要求LSE在产品设计中尽最大可行程度使用MOSA
- **2021 NDAA Section 804核心**：将MOSA要求从MDAP扩展至所有采购项目；要求建立接口存储库
- **立法演进**：2015年规划要求 → 2017年MDAP法定要求 → 2020年军种指南要求 → 2021年全项目扩展
- **实施挑战**：OSD法定要求的MOSA指南延迟发布；MTA路径缺乏MOSA政策指导；各军种指南水平不一
- **接口存储库**：2021 NDAA要求建立MOSA使能的接口存储库，用于访问接口和相关文档
- **IP策略要求**：10 USC 4401要求在采办策略中描述如何处理知识产权和技术数据权利
- **DFARS跟进**：DFARS 2021-D005 MOSA RFI（2023年11月）正在制定相关知识产权数据权利法规

## MOSA法定要求演进时间线

| 时间 | 事件 |
|------|------|
| 2016 | FY2017 NDAA Sec 805建立10 USC 4401，首次法定要求MDAP实施MOSA |
| 2019-01 | MOSA要求对新进入开发阶段的MDAP生效 |
| 2019-01 | 三军MOSA联合备忘录签署 |
| 2019-12 | FY2020 NDAA Sec 840签署，要求军种发布MOSA实施指南 |
| 2020-12 | FY2021 NDAA Sec 804签署，扩展MOSA至所有项目+接口存储库 |
| 2022-01 | OSD法定MOSA指南原定发布截止日期（实际延迟） |
| 2025-02 | DoD MOSA实施指南手册发布 |

## 实施面临的主要挑战

1. OSD法定要求的MOSA指南延迟发布（原定2022年1月）
2. MTA采办路径缺乏MOSA政策指导
3. 各军种指南水平不一（陆军最完善，海军最薄弱）
4. 接口存储库建设不完整
5. 部分项目因前期成本增加、网络安全风险、资源有限等原因未实施MOSA
