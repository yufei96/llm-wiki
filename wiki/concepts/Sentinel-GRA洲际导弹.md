# Sentinel洲际弹道导弹政府参考架构（Sentinel GRA）

## 摘要

LGM-35A Sentinel（前身为GBSD，地面战略威慑系统）是美国空军正在研发的下一代洲际弹道导弹系统，用于替代自1970年起服役的LGM-30 Minuteman III。Sentinel GRA采用模块化开放系统架构（MOSA），支持全生命周期的技术改进和能力升级。该项目由空军和Northrop Grumman（主承包商）主导，计划采购634枚导弹（400枚部署），FY2026研发预算41亿美元。Sentinel GRA位于[[开放架构层级|开放架构四层体系]]的**第2层：分系统级**，与[[NC3-GRA核指挥控制|NC3 GRA]]共同构成美国核力量的架构基础。

## 核心目标

### 替代Minuteman III的必要性

- Minuteman III自1970年服役，经过多次寿命延长，部分组件面临可靠性问题
- 2014年替代方案分析（AoA）后决定开发新系统（原名GBSD）
- 空军论证：相比寿命延长的MMIII，Sentinel能应对当前和预期威胁、维持工业基础、产生模块化武器系统、降低全生命周期成本
- 目标服役至2075年

### Sentinel GRA架构目标

1. **模块化开放系统架构**：支持组件独立升级，延长系统使用寿命
2. **数字工程**：采用数字工程技术加速设计和验证
3. **简化维护**：降低全生命周期维护成本
4. **长期适应性**：应对未来威胁变化的技术适应能力
5. **地面段简化**：Nunn-McCurdy违规后重新审视需求，使地面段"更简单"和"更经济"

## 核心特点

### 系统规模

| 指标 | 数量 |
|------|------|
| 导弹采购总数 | 634枚 |

| 部署导弹 | 400枚 |
| 导弹发射井 | 450个（新建） |
| ICBM设施 | 600+个 |
| 覆盖面积 | 40,000+平方英里 |
| 部署基地 | F.E. Warren、Malmstrom、Minot AFB |
| 作战寿命 | 至2075年 |

### 技术特征

**导弹系统**：
- 三级固体燃料火箭发动机
- 发射井发射
- 载荷：W87-0（初始）→ W87-1核弹头
- W87-0：300千吨TNT当量
- W87-1：475千吨TNT当量
- Mk21A再入飞行器（Lockheed Martin开发）

**设计哲学**：
- **数字工程技术**：全生命周期数字模型驱动
- **模块化开放系统架构**：组件可独立升级
- **简化维护**：降低运维复杂度
- **长期适应性**：支持未来威胁应对

### Nunn-McCurdy违规与重组

Sentinel项目经历了严重的成本超支：
- **2024年1月**：空军通知国会至少37%的成本增长（从1.18亿美元/枚到1.62亿美元/枚，2020年美元）
- **Nunn-McCurdy"严重"违规**：要求国防部证明项目对国家安全不可或缺、无更便宜替代方案、不能终止
- **CAPE评估**：国防部成本评估与计划评估办公室估计"合理修改"后的Sentinel成本为1410亿美元
- **里程碑B撤销**：Nunn-McCurdy审查导致里程碑B批准被撤销
- **重组方向**：重新审视需求、地面段简化、新建发射井（非复用旧井）、陆军工程兵团接管部分地面基础设施

### 国会关注议题

1. **ICBM力量必要性**：部分分析人士主张削减或取消ICBM，但2022 NPR支持Sentinel
2. **MMIII寿命延长**：部分国会议员建议再次考虑MMIII寿命延长，空军认为长期方案不可行
3. **成本与进度**：Nunn-McCurdy违规后持续关注成本和进度风险
4. **过渡风险**：MMIII与Sentinel将并行运行至少15年，GAO建议制定过渡风险管理计划
5. **弹头加载**：国会要求空军制定减少弹头加载时间的计划
6. **多弹头配置**：战略态势委员会提议考虑Sentinel多弹头配置或部分公路机动

## 与其他标准的关系

### 在开放架构层次体系中的位置

Sentinel GRA位于[[开放架构层级|开放架构四层体系]]的**第2层：分系统级**，定义洲际弹道导弹分系统的模块化架构。

### 纵向关系

```
NC3 GRA（分系统级）──指挥控制──→ Sentinel GRA
Sentinel GRA（分系统级）──可能采用──→ SOSA/OMS（设备级/平台级）
Sentinel GRA（分系统级）──数字工程──→ 模型化系统工程
```

### 横向互补

- **Sentinel GRA ↔ NC3 GRA**：核力量体系的武器投射层 ↔ 指挥控制通信层，紧密耦合
- **Sentinel GRA ↔ WOSA**：Sentinel可采用WOSA武器开放架构标准
- **Sentinel GRA ↔ DEWS RA**：定向能武器的未来可能与ICBM防御相关

### 与MOSA的关系

Sentinel强调模块化开放系统架构，但其实现面临特殊挑战：
- 核武器系统的安全性要求限制了开放程度
- 长寿命系统（至2075年）的模块化设计需要极高的前瞻性
- 地面基础设施（发射井、指挥中心）的模块化与导弹本身的模块化需要协调
- Nunn-McCurdy违规后的重组正在推动更多模块化和简化设计

### 与NC3 GRA的关系

Sentinel GRA与[[NC3-GRA核指挥控制|NC3 GRA]]是美国核力量的两大架构支柱：
- NC3 GRA确保指挥控制通信链路的可靠性和弹性
- Sentinel GRA确保武器投射系统的可靠性和适应性
- 两者必须在架构层面协调——Sentinel必须能接收并执行NC3传达的命令
- NC3的AEHF通信和SBIRS预警直接支持Sentinel的作战流程

## 笔记

- Sentinel是美国核三位一体现代化中规模最大、成本最高的单一项目，Nunn-McCurdy违规反映了地面基础设施更新的复杂性
- **模块化开放系统架构**是Sentinel设计哲学的核心——目标是支持至2075年的持续升级，这要求极高的架构前瞻性
- Boeing在2019年退出竞争使Northrop Grumman成为唯一主承包商，这引发了对竞争和成本控制的担忧
- Nunn-McCurdy违规后的重组方向（简化地面段、新建发射井）说明开放架构不仅关乎技术，也关乎项目管理
- MMIII与Sentinel至少15年的并行运行期是前所未有的挑战——需要同时维持两套系统的作战能力
- 战略态势委员会提议的多弹头配置和公路机动可能从根本上改变Sentinel的架构需求
- W87-1弹头的plutonium pit合格认证（2024年10月）是关键里程碑，但NNSA的pit生产能力仍是瓶颈
- Sentinel GRA与NC3 GRA的协调是核力量现代化成功的关键——任何一方的延误都会影响整体效能
- 陆军工程兵团接管部分地面基础设施打破了Northrop Grumman的独占，是促进竞争的实际举措
- FY2025协调立法的$2.5B"风险降低活动"反映了国会虽然支持项目但也要求更多保障
