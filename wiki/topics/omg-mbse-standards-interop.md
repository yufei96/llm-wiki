# 建模标准与互操作

## 建模标准活动

MBSE Initiative 的建模标准活动团队关注系统工程领域建模语言和工具的标准化工作，主要成果包括：

- SysML v1 和 v2 规范
- UPDM（统一 Profile for DoDAF/MODAF）
- Modelica 物理建模语言
- HLA（高层架构）仿真互操作标准

## 建模仿真互操作

建模仿真互操作（Modeling & Simulation Interoperability）挑战团队后发展为 DECO 数字生态系统团队，关注：

- 不同建模工具之间的数据交换
- 异构模型的集成与协同仿真
- 供应链中的模型互操作

## 与 SysML v2 的关系

SysML v2 的 API & Services 规范直接解决了建模互操作问题——通过标准化接口实现工具间的模型导航、查询和更新。
