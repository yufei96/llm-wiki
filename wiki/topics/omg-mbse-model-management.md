# 模型管理

## 概述

模型管理活动团队关注模型的生命周期管理，后发展为 INCOSE 工具集成与模型生命周期管理工作组。

## 核心议题

- 模型版本控制与配置管理
- 模型基线与变更管理
- 多工具环境下的模型一致性
- 模型库（Model Repository）架构

## 与 Tool Integration WG 的关系

模型管理团队的工作直接促成了 INCOSE Tool Integration and Model Lifecycle Management Working Group 的建立，继续推进工具集成和模型生命周期管理的标准制定。
